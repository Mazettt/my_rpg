<h1 align="center">My RPG<br/>Epitech projet</h1>

A RPG game made in C with CSFML library.

> Done between March, 21 2022 and May, 8 2022

## Requirements

- SFML and CSFML
- Make
- GCC

## Install

```sh
git clone git@github.com:Mazettt/my_rpg.git && cd my_rpg && make
```

## Usage

```sh
./launch_game.sh
```
