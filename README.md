<h1 align="center">My RPG<br/>Epitech projet</h1>

A RPG game made in C with CSFML library.

## Install

```sh
git clone git@github.com:Mazettt/my_rpg.git
cd my_rpg
make
```

## Usage

```sh
./launch_game.sh
```
